Real-Time Hybrid Testing

 

Cyber-physical Instrument for Real-time hybrid Structural Testing (CIRST)
Real-time hybrid testing combines simulation and physical testing to help in assessing structural integrity in large scale civil infrastructure. The RTHS test frame in the IISL allows for configurable testing of the real components in the system, while actuators impose forces determined by the numerical components.

 

Shake Tables

 

Six Degree of Freedom Shake Table
Run by six hydraulic actuators, this small-scale shake table allows motion ranging up to six degrees of freedom, including translation and rotation about the three orthogonal axes. This shake table is typically used to test small scale multi-degree of freedom building systems. The shake table operation is controlled through dSpace.

 

Single Degree of Freedom WinCon Shake Table
This shake table was constructed by Quanser Consulting and serves as the standard shake table of the UCIST consortium. The shake table operation is controlled through WinCon and MatLab.

 

Single Degree of Freedom Tele-operation Shake Table
This shake table operates in both local and remote applications, enabling joint research projects with various laboratories. The shake table operation is controlled through LabVIEW.
